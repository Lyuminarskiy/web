Описание элемента <details>

Элемент <details> используется для хранения информации, которую можно скрыть или показать по требованию пользователя. По умолчанию содержимое элемента не отображается, для изменения статуса применяется атрибут open.

Атрибуты: open

Описание элемента <summary>

Элемент <summary> указывает заголовок для элемента <details>, по которому можно щелкать для разворачивания/сворачивания информации. Элемент <summary> должен идти первым внутри <details>.
