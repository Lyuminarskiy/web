### Структура кода

#### Комментарии в коде

Комментарии используются для добавления подсказок, заметок, предложений
или предупреждений, а также предотвращения выполнения участка кода.
Они могут упростить чтение и понимание кода.

В JavaScript есть несколько способов указать комментарии в коде:

- однострочный комментарий `//...`:
  
  ```js
  // Однострочный комментарий.
  // На следующей строке необходимо снова указать две косые черты.
  ```
  
- многострочный комментарий `/*...*/`:

  ```js
  /* Многострочный комментарий распространяет
     своё действие на все строки между открывающим
     и закрывающим символами. */
  ```
  
Ссылки по теме:

- [Лексический синтаксис
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Lexical_grammar#Комментарии).

#### Команды

Код на JavaScript состоит из команд, разделённых между собой
точкой с запятой `;`. Обычно каждая команда пишется на отдельной строке.

Пример:

```js
2 + 3; // Первая команда.
3 - 1; // Вторая команда.
```

#### Числовой тип

Числовой тип един как для целых чисел, так и для чисел с плавающей запятой.

Существует несколько способов создания числа:

```js
123 // Целое число.
123.45 // Число с плавающей запятой.

// Экспоненциальная запись
3e-5 // Эквивалентно 0.00003.
1e+3 // Эквивалентно 1000.

0x10 // Шестнадцатеричная запись, эквивалентно 16.
0o10 // Восьмеричная запись, эквивалентно 8.
0b10 // Двоичная запись, эквивалентно 2. 
```

При делении на ноль можно получить специальное значение - бесконечность [`Infinity`
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Infinity):

```js
// Равняется Infinity.
1 / 0
Infinity * 1

// Равняется -Infinity.
-1 / 0
Infinity * -1
```

Также можно получить специальное значение [`NaN`
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/NaN),
в тех ситуациях, когда математическая операция не может быть совершена:

```js
// Равняется NaN.
0 / 0
'строка' * 2
NaN + 1
NaN + Infinity
```

Ссылки по теме:

- [Числа](https://learn.javascript.ru/number).

#### Строковый тип

В JavaScript любые текстовые данные являются строками, даже строка из одного 
символа.

Существует несколько способов создания числа:

- без переноса строк:

  - с помощью двойных кавычек - `"однострочная строка"`,
  - с помощью одинарных кавычек - `'однострочная строка'`,
  
- с переносами строк с помощью обратных кавычек:

  ```js
  `многострочная
      строка`
  ```

  Такие строки называются также _шаблонными_. Дополнительно в эти строки 
  можно вставлять выражения при помощи `${...}` - _выражения интерполяции_:
  
  ```js
  `Для изготовления салата нужно ${2} огурца и ${2 + 3} помидора`
  ```
  
  В результате интерполяции мы получим строку следующего вида:
  
  ```js
  "Для изготовления салата нужно 2 огурца и 5 помидора"
  ```
  
Строки могут содержать специальные символы. Для этого происходит операция
_экранирования_ с помощью символа обратной косой черты `\`. Вот некоторые 
из таких сиволов:

- `\n` - символ перевода строки,
- `\\` - символ обратной косой черты.

Пример:

```js
"Многострочная строка,\nс символом обратной косой черты.\\"
```

У каждой строки есть свойство `length`, с помощью которого можно узнать
длину строки:

```js
"Строка".length; // 6
"Очень длинная строка".length; // 20
```

- [Строки](https://learn.javascript.ru/es-string).

#### Логический тип

Логический тип представлен всего двумя значениями:

- `true` - истина,
- `false` - ложь.

#### Тип `null`

Этот тип состоит из единственного значения `null`. Это значение имеет смысл
"ничего" или "значение неизвестно".

#### Тип `undefined`

Этот тип, также как и `null`, состоит из единственного значения `undefined`.
Это значение имеет смысл "значение не присвоено".

#### Объекты

Предыдущие пять типов называют "примитивными". Объекты представляют из себя 
более сложные сущности, так как позволяют создавать сложные структуры данных.

Объект создаются с помощью использования фигурных скобок:

```js
{} // Пустой объект.
```

#### Проверка типа

Для получения типа значения в виде строки используется оператор [`typeof`
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Operators/typeof):

```js
typeof 3.14; // "number"
typeof true; // "boolean"
typeof "строка"; // "string"
typeof {}; // "object"
```

### Переменные

В JavaScript есть три способа создания переменных:

1. С помощью ключевого слова [`let`
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Statements/let),
Созданная таким образом переменная имеет свои особенности:

    - Переменную можно использовать только после создания:
    
      ```js
      a + 2;
      let a = 5; // Ошибка: нет такой переменной.
      ```
      
    - Нельзя повторно создавать переменную с тем же именем:
    
      ```js
      let b;
      let b; // Ошибка: переменная уже была создана.
      ```
    
    - Область видимости такой переменной - текущий блок `{...}` программы:
    
      ```js
      if(true) {
        let c = 3; // Переменная создаётся внутри блока if.
      }

      c + 2; // Ошибка: нет такой переменной.
      ```

2. C помощью ключевого слова [`const`
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Statements/const).
Созданная таким образом переменная представляет из себя _константу_, т.е. 
переменную, значение которой нельзя изменять:
  
    ```js
    const PI = 3.14;
    PI = 3.00; // Ошибка: нельзя перезаписывать константу.
    ```
    
    В остальном константы имеют те же свойства, что и переменные, созданные с 
    помощью ключевого слова `let`.

3. С помощью ключевого слова [`var`
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Statements/var).
Созданная таким образом переменная имеет следующие особенности:

    - Переменную можно использовать до создания, она будет равна `undefined`:
    
      ```js
      a + 2; // Ошибки нет. Выражение (a + 2), или (undefined + 2) равно NaN.
      var a = 5;
      ```
      
    - Можно повторно создавать переменную с тем же именем:
    
      ```js
      var b = 2;
      var b; // Ошибки нет, значение переменной не изменится.
      ```
    
    - Область видимости такой переменной - текущий контекст выполнения
    программы (объект или функция):
    
      ```js
      if(true) {
        var c = 3; // Переменная создаётся внутри блока if.
      }
  
      c + 2; // Ошибки нет, текщий контекст выполнения - объект window.
      ```

### Структуры данных

#### Массивы

Массив – разновидность объекта, которая предназначена для хранения
пронумерованных значений. Элементы нумеруются, начиная с нуля.

Пример работы с массивами:

```js
// Пустой массив.
let empty = [];
let empty = new Array();

let names = ["Иван", "Василий", "Александр"]; // Массив с тремя элементами.
names[0]; // Доступ к первому элементу массива - "Иван".
names[1]; // Доступ к второму элементу массива - "Василий".
names[2]; // Доступ к третьему элементу массива - "Александр".

// Замена третьего элемента массива.
// Теперь массив выглядит так: ["Иван", "Василий", "Дмитрий"]
names[2] = "Дмитрий";

// Добавление четвёртого элемента массива.
// Теперь массив выглядит так: ["Иван", "Василий", "Дмитрий", "Сергей"]
names[3] = "Сергей";
```

У каждого массива есть свойство `length`, с помощью которого можно узнать
количество элементов, хранимых в массиве:

```js
let names = ["Иван", "Василий", "Александр"];
names.length; // 3

[3.14, true, "строка", [1, 2, 3]].length; // 4
```

Ссылки по теме:

#### Объекты

Объект как структура данных представляет из себя ассоциативный массив.
Данные в объексте хранятся в формате _"свойство-значение"_.

Пример работы с объектами:

```js
// Пустой объект.
let first = {};
let second = new Object();

let user = {name: "Иван", age: 25}; // Создание объекта с двумя свойствами.

user.name; // "Иван", получение значения свойства name.
user["name"]; // "Иван", другой способ получения значения того же свойства.

user.surname; // undefined, в объекте нет свойства surname.

// Изменение значения свойства age.
// Теперь объект выглядит так: {name: "Иван", age: 33}
user.age = 33;

// Создание нового свойства surname.
// Теперь объект выглядит так: {name: "Иван", age: 33, surname: "Иванов"}
user.surname = "Иванов";

"surname" in user; // true, проверка начилия свойства surname в объекте.

// Удаляет свойство surname.
// Теперь объект выглядит так: {name: "Иван", age: 33}
delete user.surname;

"surname" in user; // false, проверка начилия свойства surname в объекте.
```

### Базовые операторы

В JavaScript есть следующие базовые типы операторов:

- Операторы присваивания:

  ```js
  x = y;    // Присваивание.
  x += y;   // Присваивание со сложением.
  x -= y;   // Присваивание с вычитанием.
  x *= y;   // Присваивание с умножением.
  x /= y;   // Присваивание с делением.
  x %= y;   // Присваивание по модулю.
  x <<= y;  // Присваивание с левым сдвигом.
  x >>= y;  // Присваивание с правым сдвигом.
  x >>>= y; // Присваивание с беззнаковым сдвигом вправо.
  x &= y;   // Присваивание с побитовым И.
  x ^= y;   // Присваивание с побитовым исключающим ИЛИ.
  x |= y;   // Присваивание с побитовым ИЛИ.
  ```

- Операторы сравнения:

  ```js
  x == y;  // Равно.
  x != y;  // Не равно.
  x === y; // Строго равно.
  x !== y; // Строго не равно.
  x > y;   // Больше.
  x >= y;  // Больше или равно.
  x < y;   // Меньше.
  x <= y;  // Меньше или равно.
  ```
  
- Арифметические операторы:
  
  ```js
  x + y;  // Сложение.
  x - y;  // Вычитание.
  x * y;  // Умножение.
  x ** y; // Возведение в степень.
  x / y;  // Деление.
  x % y;  // Остаток от деления.
  +x;     // Унарный плюс.
  -x;     // Унарный минус.
  ++x;    // Префиксный инкремент.
  --x;    // Префиксный декремент.
  x++;    // Постфиксный инкремент.
  x--;    // Постфиксный декремент.
  ```
  
- Битовые операторы:
  
  ```js
  x & y;   // Побитовое И.
  x | y;   // Побитовое ИЛИ.
  x ^ y;   // Побитовое исключающее ИЛИ.
  x << y;  // Сдвиг влево.
  x >> y;  // Сдвиг вправо с переносом знака.
  x >>> y; // Сдвиг вправо с заполнением нулями.
  ~x;      // Побитовое НЕ.
  ```

- Логические операторы:
                                 
  ```js
  x && y; // Логическое И.
  x || y; // Логическое ИЛИ.
  !x;     // Логическое НЕ.
  ```

В одном выражении может присутствовать множество различных операторов.

### Преобразование типов

Когда в выражениях принимают участие операнды разных типов, в дело вступает
система преобразования типов JavaScript.

Всего есть три преобразования:

- _Строковое преобразование_ - происходит, когда требуется представление
чего-либо в виде строки:

  ```js
  // Явное преобразование.
  String(123); // "123"
  String(null); // "null"

  // Неявное преобразование.
  true + "test"; // "truetest"
  "123" + undefined // "123undefined"
  ```

- _Численное преобразование_ - происходит в математических функциях и
выражениях, а также при сравнении данных различных типов (кроме `===` и `!==`):

  ```js
  // Явное преобразование.
  Number("123"); // 123
  Number("строка");  // NaN
  Number(true); // 1
  
  // Неявное преобразование.
  +"123"; // 123
  +undefined; // NaN
  true / false; // Infinity
  
  // Сравнение разных типов приводит к численному преобразованию
  "" == false; // true
  "1" == true; // true
  ```

- _Логическое преобразование_ - происходит в логическом контексте
(операторы ветвления) и при применении логических операторов:

  ```js
  // Явное преобразование.
  Boolean(123); // true
  Boolean(NaN); // false
  Boolean("строка"); // true
  Boolean(""); // false
  Boolean(null); // false
  Boolean({}); // true

  // Неявное преобразование.
  !!"0"; // true
  !!""; // false
  !!undefined; // false
  !![]; // true
  !![1, 2, 3]; // true
  ```

- [Грамматика и типы

### Объект `window`

Объект [`window`](https://developer.mozilla.org/ru/docs/Web/API/Window) -
контекст выполнения программы по умолчанию. Любая переменная, созданная
с помощью ключевого слова `var`, становится также свойством объекта `window`:

```js
// Создаём переменную prop.
var prop = "Привет, мир!";

// В объект window добавилось одноимённое свойство.
window.prop; // "Привет, мир!"

// Мы можем обращаться к свойствам и методам объекта `window` неявно.
prop; // "Привет, мир!"
```

Объект `window` содержит набор полезных методов:

```js
// Отображает диалоговое окно с сообщением и кнопкой OK.
// Ещё раз: мы можем обращаться к свойствам и методам объекта `window` неявно.
window.alert("Привет, мир!");
alert("Привет, мир!");

// Отображает диалоговое окно с сообщением и кнопками OK и Cancel.
// Возвращает true или false, в зависимости от нажатой пользователем кнопки.
let answer = window.confirm("Вам нравится наш веб-сайт?");
answer = confirm("Вам нравится наш веб-сайт?");

// Отображает диалоговое окно с сообщением с запросом на ввод текста.
// Возвращает введённый пользователем текст, либо значение по умолчанию.
let text = window.prompt("Пожалуйста, напишите ваш отзыв:", "Всё отлично!");
text = prompt("Пожалуйста, напишите ваш отзыв:", "Всё отлично!");

// Возвращает целое число на основе строки
// в соответствии с указанным основанием системы счисления.
let int = window.parseInt("FF", 16); // 256
int = parseInt("строка", 8); // NaN

// Возвращает число с плавающей запятой на основе строки.
let float = window.parseFloat("3.14"); // 3.14
float = parseFloat("строка"); // NaN
```

Ссылки по теме:

- [Взаимодействие с пользователем: alert, prompt, confirm
](https://learn.javascript.ru/uibasic).

### Операторы ветвления

JavaScript поддерживает три оператора ветвления:

- _Условный оператор_ - возвращает значение в зависимости от условия:

  ```js
  // Запрашиваем у пользователя его возраст.
  let age = +prompt("Сколько вам лет?", "18");

  // На основании возраста выбираем сообщение.
  let message = age < 30 ? "Привет!" : "Здравствуйте";
  alert(message);

  // Условные операторы можно объединять в цепочку.  
  let message =
    age < 10
      ? "Привет, малыш!"
      : age < 30
        ? "Привет!"
        : "Здравствуйте!";
  alert(message);
  ```
  
- _Оператор `if..else`_ - выполняет тот или иной блок кода в зависимости от 
условия:

  ```js
  // Запрашиваем у пользователя его возраст.
  let age = +prompt("Сколько вам лет?", "18");
  
  // На основании возраста выбираем сообщение.
  if(age < 30) {
    alert("Привет!");
  }
  else {
    alert("Здравствуйте");
  }
  
  // Операторы if..else можно объединять в цепочку.  
  if(age < 10) {
    alert("Привет, малыш!");
  }
  else if(age < 30) {
    alert("Привет!");
  }
  else {
    alert("Здравствуйте!");
  }
  ```
  
- _Оператор `switch`_ - выполняет тот или иной блок кода в зависимости от 
значения выражения:

  ```js
  // Запрашиваем у пользователя текущий месяц.
  let month = +prompt("Какой сейчас месяц?");

  // На основании месяца выводим текущее время года.
  // Все варианты проверяются на строгое равенство значение month.
  switch(month) {
    case "Декабрь":
    case "Январь":
    case "Февраль":
      // Если совпал один из вариантов, то начинает выполняться блок когда
      // от соответствующего варианта case до ближайшей инструкции break.
      alert("Сейчас зима");
    
      // Выходит из оператора switch.
      break;
    
    case "Март":
    case "Апрель":
    case "Май":
      alert("Сейчас весна");
      break;
    
    case "Июнь":
    case "Июль":
    case "Август":
      alert("Сейчас лето");
      break;
    
    case "Сентябрь":
    case "Октябрь":
    case "Ноябрь":
      alert("Сейчас осень");
      break;
    
    // Если не один вариант не совпал, то выполняется ветка default.
    default:
      alert("Такого месяца не существует!");
      break;
  }
  ```

Ссылки по теме:

- [Условные операторы: if, '?'](https://learn.javascript.ru/ifelse).
- [Конструкция switch](https://learn.javascript.ru/switch).

### Замена условных инструкций

В фрагменте программы замените оператор:

1. ... [`if..else`
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Statements/if...else)
на [условный оператор
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Operators/Условный_оператор):

```js
var a = 2, b = 3, result;

if (a + b < 4) {
  result = "Верно";
}
else {
  result = "Неверно";
}
```

2,3. ... [`if..else`
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Statements/if...else)
на [условный оператор
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Operators/Условный_оператор)
и (отдельно) оператор [`switch`
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Statements/switch):

```js
var login = prompt("Введите логин", ""), message;

if (login === "Сотрудник") {
  message = "Привет";
}
else if (login === "Директор") {
  message = "Здравствуйте";
}
else if (login === "") {
  message = "Логин не указан";
}
else {
  message = "Неизвестный пользователь";
}

alert(message);
```

4. ... [`switch`
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Statements/switch)
на оператор [`if..else`
](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Statements/if...else):

```js
var browser = prompt("Какой у вас браузер?", ""), message;

switch (browser) {
  case "IE":
    message = "Давай, до свидания!";
    break;

  case "Chrome":
  case "Firefox":
  case "Safari":
  case "Opera":
  case "Edge":
    message = "Эти браузеры поддерживаются";
    break;    

  default:
    message = "Браузер неизвествен";
}

alert(message);
```
